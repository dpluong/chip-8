**Chip-8 CMPT 276 Spring 2019 Team 8**

A Chip-8 emulator, visualizer, OTHER TOOL, and two Chip-8 games. Made in Javascript.

---

## Install

1. Clone the resp
2. `npm install`
3. `npm run build`


---

## Testing

To test the code, do `npm test`
